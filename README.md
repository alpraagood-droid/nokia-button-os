# Nokia Button OS

Minimal operating system concept for keypad phones

## Target Devices
- Nokia 3310
- Nokia 1100
- Nokia 105

## Architecture
- Bare-metal (no Android / no Linux)
- Assembly-based kernel (2KB)
- Button-driven UI
- Low power management

## Apps
- Phone
- SMS
- Lightweight Browser (concept)
- Social Lite UI (Facebook Lite / TikTok Lite concept)
